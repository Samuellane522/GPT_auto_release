Install

One-click setup via userscript bootstrap

Install the Auto Prompter bootstrap userscript and start securely loading encrypted releases.

1) Install Tampermonkey

Chrome / Edge / Firefox — install the official Tampermonkey extension.

2) Install the Bootstrap userscript

Click the button; Tampermonkey will prompt you to install:

Install bootstrap.user.js 
https://samuellane522.github.io/GPT_auto_release/beta-release/current/boot/auto-prompter-bootstrap.user.js

3) Trust but verify

We publish an encrypted bundle and a policy file with SRI.

  • enc.bin: https://samuellane522.github.io/GPT_auto_release/download/beta/auto-prompter-enc.bin
  • SRI: sha256-wizWGcOh/2A5QoUFnlD5dgyzofVAzYLSLxuURFj2n+8=

Prefer the CLI? Use the verification page.
