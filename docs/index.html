Install

  1. Install Tampermonkey for your browser.
  2. Open the bootstrap link and accept the userscript prompt:
    https://samuellane522.github.io/GPT_auto_release/beta-release/current/boot/auto-prompter-bootstrap.user.js
  3. Refresh ChatGPT. The bootstrap securely fetches and decrypts your bundle.

Tip: You can always re-install the bootstrap to update it. The encrypted payload updates independently per release.

How it works

Encrypted bundle

Your main userscript ships as enc.bin, decrypted in-browser by the bootstrap.

Remote key

The AES key lives at key.json, so you can rotate credentials without changing the bootstrap.

SRI policy

We publish Subresource Integrity hashes so you can verify the exact bits served by Pages.

Versioned docs

Every ship snapshots docs under /docs/v/<build-id> for easy back-references.

Troubleshooting

  • No prompt changes? Ensure Tampermonkey is enabled for chat.openai.com/chatgpt.com and refresh.
  • SRI mismatch? Re-pull the repo and re-ship; verify policy.json and checksums.txt are updated.
  • Cache issues? Hard refresh the site hosting the bootstrap and the enc bundle.
  • Domain lock? If configured, features only run on allowed hosts.

FAQ

Can I pin to a specific build?
Yes. Use the versioned docs under /docs/v/<build-id> and the corresponding artifacts from that build.
How often should I rotate the key?
We recommend at most every 30 days (configurable by your pipeline). Rotating doesn’t require changing the bootstrap.
How do I uninstall?
Remove the bootstrap from Tampermonkey and clear any site data if desired. No files are written outside the extension.
